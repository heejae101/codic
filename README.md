# codic
코딩 백과사전 사이트(2차 프로젝트 조장 채희재, 서기 김주연, 팀원 유현호)

### 사이트
[사이트 바로가기](http://codicblog.site/)

### 개요
 - 코딩 수업을 들을때에는 이해가 잘 되고 쉬웠던 코딩들도 프로젝트가 시작하게 되면 다 까먹기 마련이였습니다.
 - 저희 사이트는 배웠던 코딩내용을 기록하고 까먹을때 마다 다시 볼 수 있는 사이트로 제작했습니다.

### 제작기간
 - 2023.07.03 ~ 2023.07.21
